# QA Automation Challenge - DemoQA

## 📌 Overview
This project contains automated end-to-end tests for the DemoQA application using Cypress and JavaScript.

The objective is to demonstrate test automation skills, test design, and best practices for UI testing.

---

## 🚀 Tech Stack
- Cypress
- JavaScript
- Page Object Model (POM)

---

## 📁 Project Structure

cypress/
  e2e/            → Test specifications
  fixtures/       → Test data
  support/
    commands.js   → Custom commands
    e2e.js        → Global setup
    locators/     → Centralized selectors
    pages/        → Page Objects
    utils/        → Helper functions
